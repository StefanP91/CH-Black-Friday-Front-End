ВОВЕД ВО ФРОНТ-ЕНД ПРОГРАМИРАЊЕ
Запознавање со основите на веб-развојот, улогата на фронт-енд девелоперот, логиката зад веб-технологиите и начинот на кој прелистувачите ги прикажуваат веб-страниците. Идеален модул за поставување здрави темели пред да преминеш на HTML, CSS и JavaScript.
HTML
Ќе ја научиш структурата на веб-страниците, елементите, таговите, семантиката и најдобрите практики за чист и организиран markup. HTML е основата врз која се гради целиот фронт-енд.
CSS
Стилови, layout системи (Flexbox, Grid), responsive дизајн, анимации и визуелни ефекти. Овој модул ќе ти помогне да ја трансформираш „голата“ HTML структура во модерни, атрактивни и функционални веб-интерфејси.
BOOTSTRAP
Ќе научиш како да изградиш брзи, responsive веб-страници користејќи го најпопуларниот CSS фрејмворк. Компоненти, layout системи, навигации, картички и готови UI елементи кои штедат време и го забрзуваат процесот.
JAVASCRIPT
Клучниот модул за секој фронт-енд девелопер. Ќе ја совладаш логиката, променливи, функции, DOM манипулација, event listeners, API комуникација и реално програмирање. JavaScript е јазикот што ги прави веб-страниците живи и интерактивни.
REACT
Современ front-end framework користен од компаниите ширум светот. Ќе научиш компоненти, props, state, hooks, routing, работа со API, управување со податоци и креирање на реални веб-апликации. Реактивно програмирање според актуелните индустриски стандарди.
АКАУНТ МЕНАЏМЕНТ
Овој модул те подготвува за работа со клиенти во реален проектен процес. Ќе научиш комуникација, планирање, управување со барања, организирање работа и презентирање решенија како вистински IT професионалец.
КОМУНИКАЦИСКИ ВЕШТИНИ ВО IT
Фокус на софт-скилови што се неопходни за работа во тим — техничка комуникација, remote работа, давање и примање feedback, презентациски вештини и работа со програмерски тимови. Идеален модул за кариера во динамична IT средина.
